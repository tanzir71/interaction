## Shared aesthetic system (referenced by every item — memorize)

**Palette (site tokens + functional colors):**

| Token | Value | Use |
|---|---|---|
| `bg0` | `#0a0a0b` | stage background |
| `bg1` | `#101012` | panels, cards |
| `bg2` | `#161619` | raised/hover surfaces, input wells |
| `line0` | `#232327` | default 1px hairline borders |
| `line1` | `#2e2e34` | hover/active borders, dividers |
| `txt0` | `#ececef` | primary text, data values |
| `txt1` | `#9b9ba3` | secondary text |
| `txt2` | `#5c5c66` | labels, captions, disabled |
| `acc` | `#a78bfa` | THE accent. Interactive highlights, scan effects, active states |
| `ok` | `#4ade80` | success/positive only |
| `warn` | `#fbbf24` | warnings only |
| `err` | `#f87171` | errors/negative only |
| `info` | `#67e8f9` | secondary data hue (charts, second series) — sparingly |

Never introduce other hues. Gradients: only between a color and its transparent self, or `bg` shades.

**Type:** `"JetBrains Mono", monospace` everywhere. Labels/captions: 10px, uppercase, `letter-spacing: 0.08em`, `txt2`. Data values: 12–13px, `txt0`. Big numerals: 26–34px, weight 500. No other fonts, no italics.

**Surfaces:** panels are `bg1` with 1px `line0` border, `border-radius: 10px` (chips/pills 999px, hard-tech FUI panels 4px). Inner wells `bg2`, inset border. No drop shadows for elevation on dark — elevation = lighter surface + brighter border. Glow only as feedback: `box-shadow: 0 0 12px <color at ~35% alpha>`.

**Motion defaults (use unless an item overrides):**
- `ease-out-soft` = `cubic-bezier(0.22, 1, 0.36, 1)` — entrances, expansions
- `ease-in-out` = `cubic-bezier(0.65, 0, 0.35, 1)` — position swaps, morphs
- spring = GSAP `elastic.out(1, 0.65)` or manual spring stiffness 170 / damping 22 — playful settles only
- micro-interactions 160–240ms; entrances 500–800ms; staggers 30–60ms/item; hover-in fast, hover-out ~1.5× slower
- cursor-follow lerp: `pos += (target - pos) * 0.12` per frame (0.08 = heavier, 0.18 = snappier)
- radial falloffs: `influence = max(0, 1 - dist/radius)` then square it for a soft knee
- respect `prefers-reduced-motion`: kill loops/parallax, keep state changes as instant fades

**Texture:** film grain = canvas noise at 2–3% opacity, regenerated at ~8fps, `mix-blend-mode: overlay`. Scanlines (FUI only) = `repeating-linear-gradient(0deg, transparent 0 2px, rgba(255,255,255,0.02) 2px 3px)`. Vignette = radial gradient to `rgba(0,0,0,0.35)` at edges.

## Reminders that bite in this batch

- No network, no image/font/geodata assets — all source pixels procedural (canvas gradients, noise, math shapes). Maps are *fake geography*: seeded-random polygons, noise coastlines, hand-rolled point arrays. It must read as a map, not be one.
- Sound only where noted: WebAudio-synthesized, gesture-gated, visible mute.
- `seen:` must credit the popularizer named in the item (or a real reference you verify).
- No backticks or `${` inside `html`/`css`/`js`/`prompt` template literals — concatenation and `.join('\n')` only.

### Agent & AI UI (new file `js/demos.agent.js`)

**Category look:** this is product UI, not FUI — softer. Panels `bg1` radius 10px, 13px text, generous 16px padding. `acc` = agent activity color (streams, pulses, active nodes). `ok`/`err` reserved for pass/fail. Streaming text is the hero: get the cadence right — LLM output arrives in bursts of 1–4 tokens with 30–90ms gaps and occasional 300ms stalls, NOT a metronome typewriter.

Simulation: bursts every 1.5–3s add 800–4k tokens; sparkline (last 30 samples, 1.5px `info` line) pushes left with 300ms tween; cost `$0.412` count-rolls per digit (slot-style, 250ms). At 70% ring turns `warn`, at 90% `err` + gauge pulse (glow 0→30%, 1s loop) + a `COMPACTING…` chip appears and the ring animates back down to 35% over 1.2s — then the cycle continues. Seen: Jeet Nirnejak's Token Usage component
- [ ] `agent-deploy-pipeline` — stage rail. 4 nodes (BUILD, TEST, PREVIEW, PROD) as 32px circles joined by 2px `line0` connectors. Active node: `acc` ring rotating (conic-gradient mask, 1.2s/rev), connector to it fills `acc` left→right as a progress proxy.
