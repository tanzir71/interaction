What it deliberately does not do

  • Snippets are vanilla-first. Some declare GSAP or Lenis from a CDN when the effect needs them.
  • LLM prompts are implementation specifications, not prebuilt React, Vue, or Svelte ports.
  • A demo is a reference, not a promise that every pattern fits your production accessibility and performance budget unchanged.

FAQ

What is the license?

MIT. Use, modify, and redistribute the code under the repository license.

Can I use a pattern commercially?

Yes. The MIT license permits commercial use. Review any external library license declared by the pattern too.

How do I add a pattern?

Register one object in the matching js/demos.*.js file with a unique ID, prefixed root class, demo strings, and a detailed prompt, then run the roadmap and browser verifiers.

Are the prompts framework ports?

No. They describe structure, math, timing, easing, performance, and accessibility so your agent can produce a native implementation in your stack.
